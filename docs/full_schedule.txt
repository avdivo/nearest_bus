1. Расписание для одноименных остановок отправления 
    - найти объекты всех одноименных остановок отправления

    Представить информацию:
    Номер автобуса + (маршрут) + расписание

    Пример:
    Остановка "Дом культуры"

    Автобус №3 (Социалистическая - Мясокомбинат):
    время 1, время 2, время 3, время 4, ...

    Автобус №3 (Мясокомбинат - Социалистическая):
    время 1, время 2, время 3, время 4, ...

    Алгоритм подготовки данных
    - Получить все записи времени отправления автобуса 
    на выбранных остановках в нужный день недели (из Schedule)
    - Собрать словарь по автобусам из полученных данных отсортированный
    по номерам (для сортировки автобусов используй utils/sorted_buses.py). Для каждого автобуса словарь с остановками отправления 
    отсортированный по их id. Для каждой остановки список временных меток,
    отсортированный в порядке возрастания через генератор time_generator 
    с параметрами: (список времени, 3 часа утра, 1440).
    {
        автобус: {
            остановка: [
                cсписок временных меток
            ],
            ...
        },
        ...
    }

    Преобразование
    - Перебираем словарь по автобусам
    - Для каждой остановки получаем маршрут 
    (из Router название начальной и конечной остановки через тире). 
    У которого в Order есть эта остановка. Остановка 
    (в Order остановка с одним из одного или двух маршрутов этого автобуса)
     может встретиться для маршрута один раз, если встречается 
     дважды или больше нужно выкинуть ошибку.
    - Меняем в исходном словаре остановки на маршруты (текстовые).

Вернуть словарь:
    {
        автобус (obj): {
            маршрут (str): [
                cсписок временных меток (obj)
            ],
            ...
        },
        ...
    }
